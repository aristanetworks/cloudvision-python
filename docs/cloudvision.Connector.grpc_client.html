class cloudvision.Connector.grpc_client.grpcClient.GRPCClient(grpcAddr: str, *, certs: Optional[str] = None, key: Optional[str] = None, ca: Optional[str] = None, token: Optional[str] = None, tokenValue: Optional[str] = None)[source]

Bases: object

GRPCClient implements the protobuf client as well as its methods. grpcAddr must be a valid apiserver address in the format <ADDRESS>:<PORT>. certs, if present, must be the path to the cert file. key, if present, must be the path to .pem key file. ca, if present, must be the path to a root certificate authority file. token, if present, must be the path a .tok user access token. tokenValue, if present, is the value of the actual token. Cannot be set with token
